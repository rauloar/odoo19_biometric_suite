# Installation

## Public modules

Clone the public repositories into your Odoo custom addons path:

- `odoo19_biometric_core`
- `odoo19_biometric_fake`

Example:

```bash
git clone https://github.com/rauloar/odoo19_biometric_core.git
git clone https://github.com/rauloar/odoo19_biometric_fake.git
```

## Commercial drivers

Private driver repositories require a valid license and access granted to your GitHub account.

- `odoo19_biometric_zkteco`
- `odoo19_biometric_hikvision`
- `odoo19_biometric_import`

After cloning all allowed modules, update your `addons_path` and upgrade/install the modules in Odoo.
